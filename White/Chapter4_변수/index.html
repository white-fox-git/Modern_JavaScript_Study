- 실행 컨텍스트 : 자바스크립트 엔진이 소스코드를 평가하고 실행하기 위해 필요한 환경을 제공하고 코드의 실행결과를 관리하는 영역

변수 선언의 실행 시점과 호이스팅

호이스팅된 변수 :

✔ Check

변수 선언문보다 변수를 참조하는 코드가 앞에 있지만 JS코드는 인터프리터 언어임에도 불구하고 참조 에러가 발생하지 않고 undefine을 출력한다.

- 변수 선언이 소스코드가 한 줄씩 순차적으로 실행되는 시점(런타임)이 아닌 그 이전 단게에서 실행되기 때문

- 자바스크립트 엔진은 변수 선언을 포함한 모든 선언문을 소스코드에서 찾아내 먼저 실행한다.

- 호이스팅 : 선언문이 코드의 선두로 끌어 올려진 것 처럼 동작하는 자바스크립트 고유의 특징

- var, let, const, function, function*, class등 모든 선언문이 런타임 이전 단계에서 먼저 실행된다.
